# Planning documents

Internal curriculum drafts, kept for reference. **None of these describe the site as it currently ships.** They were all written in March 2026 and have not been updated since.

Read `content/module-N/` for what is actually live.

## Status

The platform ships **7 modules**. These documents propose a **14-module** curriculum. The two diverged:

- **Modules 8 through 14 were never built.** The lesson drafts here are the only version of that material.
- **Module 7 was built, but not from this plan.** `PHASE2-MODULE7.md` proposes *"How Apps Talk to Each Other (APIs)"*. The module that actually shipped as `content/module-7` is *"AI Anxiety Is Normal"*, which is unrelated content. The slot was reused.

So the module numbering in these filenames no longer maps to the live curriculum. Treat the numbers as draft ordering, not as a reference to any shipped module.
